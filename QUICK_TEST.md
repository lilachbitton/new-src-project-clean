# מדריך בדיקה מהירה 🔍

## לבדיקה מיידית:

### 1. התקנה והפעלה:
```bash
cd C:\Users\user\Documents\GitHub\new-src-project-clean
npm install
npm run dev
```

### 2. פתח בדפדפן:
http://localhost:3000

### 3. מה לבדוק:

#### בדף הבית:
- [x] נטען מהר ונראה טוב
- [x] יש כפתור "התחל עכשיו"
- [x] כל הטקסט בעברית

#### בבונה הצעות מחיר:

### 4. בדיקת Console:
פתח F12 ותבדוק שיש הודעות:
```

✅ API: מחזיר XX מוצרים
🔄 API: מקבל בקשה לטעינת מארזים פעילים

```

### 5. בדיקת שגיאות:
אם יש שגיאות, בדוק:
- [ ] .env.local קיים עם הטוקן
- [ ] npm install בוצע
- [ ] השרת רץ על פורט 3000
